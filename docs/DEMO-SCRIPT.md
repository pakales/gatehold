## Story contract

The video must preserve these boundaries:

- Scene A is a clean deterministic admit.
- Scene B is a model-raised semantic hold; GPT-5.6 never grants clearance.
- Scene C is deterministic host-capacity/FIFO waiting.
- Scene D is a clean finish: proven owned cleanup completes before the next
  lane clears.
- Process-group, dev-server, dedicated browser-profile, and port cleanup are
  provenance-gated.
- Ambiguous cleanup stays quarantined and retains authority.
- A pre-booted exact simulator UDID is external and never touched.
- Gatehold persists boot intent first and shuts down only an exact simulator it
  then booted and positively confirmed as owned.
- Ambiguous simulator boot intent stays quarantined with zero guessed shutdown.
- Public telemetry is synthetic replay data, not the viewer's workstation.
